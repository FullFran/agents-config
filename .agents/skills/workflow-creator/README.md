# Workflow Creator Skill ⚡

Automatiza procesos repetitivos mediante comandos slash personalizados.

## ¿Para qué sirve?
Permite formalizar secuencias de pasos complejos en un solo comando que la IA puede ejecutar de forma autónoma.

## Capacidades:
- **Automatización**: Define flujos multi-paso para el agente.
- **Optimización**: Permite el uso de `// turbo` para ejecuciones rápidas sin confirmación.
- **Integración**: Los nuevos workflows se convierten en comandos `/slash` reconocibles por OpenCode y Antigravity.

## Cómo usarla:
Pídele a tu agente o usa el comando slash:
> `/create-workflow`
> "Crea un workflow para automatizar el despliegue a staging"
